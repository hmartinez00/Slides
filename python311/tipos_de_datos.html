La función type

Determinando el tipo de dato

Entrada:


							print(type('Hellow, World!'))
							print(type(5))
							print(type(5.5))
							...

Salida:


							class 'str'
							class 'int'
							class 'float'
							...

  1. `len()` : Devuelve la longitud de la cadena.
  2. `lower()` : Convierte la cadena a minúsculas.
  3. `upper()` : Convierte la cadena a mayúsculas.
  4. `capitalize()` : Convierte el primer carácter de la cadena a mayúscula y el resto a minúsculas.
  5. `title()` : Convierte el primer carácter de cada palabra en la cadena a mayúscula y el resto a minúsculas.
  6. `strip()` : Elimina los espacios en blanco al principio y al final de la cadena.
  7. `split()` : Divide la cadena en una lista de subcadenas utilizando un delimitador especificado.
  8. `replace()` : Reemplaza una subcadena por otra en la cadena.
  9. `find()` : Devuelve la posición de la primera aparición de una subcadena en la cadena.
  10. `startswith()` : Verifica si la cadena comienza con una subcadena especificada.
  11. `endswith()` : Verifica si la cadena termina con una subcadena especificada.
  12. `join()` : Une una lista de cadenas utilizando la cadena original como separador.
